### **1、需求分析**

本实验的任务是设计并实现一个程序，给定一个只缺少左括号的表达式（例如：`1+2)*3-4)*5-6)))`），输出**补全左括号后的完整中序表达式**（例如：`((1+2)*((3-4)*(5-6)))`）。

- **输入形式**：一个合法的只缺左括号的中序表达式。
    
- **输入约定**：
    
    - 表达式中操作数为正整数（单个或多个数字组成）
        
    - 运算符为 `+`、`-`、`*`、`/` 四种之一
        
    - 表达式只缺左括号，不缺右括号，且括号结构是正确的
        
- **输出形式**：一个合法的、完整补全左括号的中序表达式

- **测试数据**：
    
    - 正确输入：`1+2)*3-4)*5-6)))` → 输出：`((1+2)*((3-4)*(5-6)))`
        
    - 正确输入：`))` → 输出：`(())`
        
    - 错误输入：含非法字符如 `1+2)*@3)` → 处理时会被忽略非法字符
        
    - 错误输入：含多余运算符如 `3+4)**5-6))` → 不在本程序处理范围内
        

---

### **2、概要设计**

- **基本思路**：
    
    - 使用两个栈 `ops`（操作符栈）和 `vals`（操作数栈，存放字符串）
        
    - 遍历表达式字符：
        
        - 若为数字，压入 `vals`
            
        - 若为运算符，压入 `ops`
            
        - 若遇到 `)`，从 `vals` 弹出两个操作数，从 `ops` 弹出一个运算符，构建新的字符串 `(val1 op val2)`，再压回 `vals`
            
    - 最终 `vals` 栈顶即为补全括号后的表达式。
        
- **主要数据结构**：
    
    ```c
    typedef struct Stack {
        char s[MAX][MAX] // 二维字符数组模拟栈
        int top
    } stack
    ```
    
- **模块调用关系**：见下函数调用关系

### **3、详细设计**

- **伪代码**（核心逻辑）：
    
    ```
    初始化两个栈 ops 和 vals
    遍历输入字符串 expr：
      如果是数字：
        读取完整数字，压入 vals
      如果是运算符：
        压入 ops
      如果是右括号：
        从 vals 弹出两个操作数 val1, val2
        从 ops 弹出一个运算符 op
        拼接成 "(val1 op val2)"，压入 vals
    输出 vals 栈顶元素，即最终表达式
    ```
    
- **函数调用关系图**：
    
    ```
   +-----------------+
   |     main()      |
   +-----------------+
	  |      |     |
  init_Stack push  pop
    ```

### **4、调试分析报告**

- **遇到的问题**：
    
    - 无法处理多位数字（如 `12+34)`
        
    - 操作数或操作符栈为空时的弹出操作可能导致段错误，无法处理 `))`
        
- **解决方法**：
    
    - 增加 `while isdigit(expr[i])` 提取完整数字
        
    - 严格检查栈边界条件
        ```c
        if (!isEmpty(&vals)) strcpy(val2, pop(&vals))
        if (!isEmpty(&vals)) strcpy(val1, pop(&vals))
        if (!isEmpty(&ops)) strcpy(op, pop(&ops))
        ```
- **时空复杂度分析**：
    
    - 时间复杂度：每次构造新的子表达式都要使用 `snprintf` 拼接字符串，最坏时间复杂度为O(n²)
        
    - 空间复杂度：O (n²)，每一个栈定义了一个 `char s[MAX][MAX]`
        
- **改进设想**：
    - 使用链表栈代替数组栈可节省空间和时间到O(n)
        
    - 增加对不合法输入表达式的判断
        
- **体会**：
    
    - 加深了对表达式结构的理解
        
    - 掌握了栈的实际应用
        
    - 提升了处理字符串与模拟栈操作的能力。
