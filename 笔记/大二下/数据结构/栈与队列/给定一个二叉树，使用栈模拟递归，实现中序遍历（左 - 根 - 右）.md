---

### 📌 问题描述：  
给定一个二叉树，使用**栈**模拟递归，实现**中序遍历（左 - 根 - 右）**，打印所有节点的值。

---

### 🌲 递归实现（参考）：

```c
void inorder(TreeNode* root) {
    if (root == NULL) return;
    inorder(root->left);
    printf("%d ", root->val);
    inorder(root->right);
}
```

---

### 🧠 非递归：用栈模拟中序遍历

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct TreeNode {
    int val;
    struct TreeNode* left;
    struct TreeNode* right;
} TreeNode;

// 栈结构
#define STACK_MAX 100
typedef struct {
    TreeNode* data[STACK_MAX];
    int top;
} Stack;

void push(Stack* s, TreeNode* node) {
    if (s->top < STACK_MAX - 1) {
        s->data[++s->top] = node;
    }
}

TreeNode* pop(Stack* s) {
    if (s->top >= 0) {
        return s->data[s->top--];
    }
    return NULL;
}

int isEmpty(Stack* s) {
    return s->top == -1;
}

void inorderTraversal(TreeNode* root) {
    Stack stack;
    stack.top = -1;
    TreeNode* curr = root;

    while (curr != NULL || !isEmpty(&stack)) {
        // 先一路向左压栈
        while (curr != NULL) {
            push(&stack, curr);
            curr = curr->left;
        }

        // 弹出并访问
        curr = pop(&stack);
        printf("%d ", curr->val);

        // 转向右子树
        curr = curr->right;
    }
}
```

---

### 🔍 讲解一下流程：

1. 从根节点一路向左，把路径上的节点全压栈。
2. 遇到最左节点后开始出栈，每出一个节点就打印它，然后转向它的右子树。
3. 再对右子树重复上述过程。

这就是把递归过程完整地模拟了一遍，用了我们自己实现的**显式栈结构**，没有系统栈参与。
